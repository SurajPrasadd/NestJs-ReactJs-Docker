Docker:
docker-compose up -d

docker compose down
docker compose up -d --build

Container Terminal :
docker exec -it backend bash
docker exec -it frontend bash

Execute the SQL file
cat ./db/init.sql | docker exec -i postgres psql -U postgres -d appdb

TRUNCATE TABLE public.categories,
               public.business,
               public.catalogs,
               public.pricing,
               public.contracts

Connect To postgres DB:
docker exec -it <container name>  psql -U postgres -d appdb
